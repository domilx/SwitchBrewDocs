SD Card Setup

Step 1: Downloading Software
----------------------------

For the necessary software, you will need to download the following:
 - [Atmosphere](https://github.com/Atmosphere-NX/Atmosphere/releases) (the latest release) and extract it to the root of your SD card, keep the fusee payload.bin as it is the payload you will need every time you want to boot into CFW.
 - [AIO Updater](https://github.com/HamletDuFromage/aio-switch-updater/releases) This software is used to update your CFW and homebrew applications. Extract the contents of the zip file to the root of your SD card.
 - [Tinfoil Installer](https://tinfoil.media/repo/Tinfoil%20Self%20Installer%20%5B050000BADDAD0000%5D%5B16.0%5D%5Bv2%5D.zip) Tinfoil is an application to install backups of gamers. Extract the contents of the zip file to the root of your SD card.
 - [NX Themes Installer](https://github.com/exelix11/SwitchThemeInjector/releases) NX Themes Installer is an application to install custom themes on your switch. Extract the contents of the zip file to the root of your SD card. Put the NRO somewhere in the `switch` folder.
 - [Hekate](https://github.com/CTCaer/hekate/releases) You will need hekate at first to setup the emmuNAND and once that is done you will go back to fusee.
